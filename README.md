# Glyph Recall

**A calm, free short-term memory trainer — no ads, no accounts, no pressure.**

Glyph Recall is a simple browser game built to train and test short-term memory, working memory, and cognitive recall. Memorize a small set of glyphs (a symbol + a word), then find them again in a grid — while the game gets a little harder each round.

## How it works

1. **Memorize.** You're shown a few glyphs — things like 🦓 Zebra or 🔑 Key — and you take as long as you need to commit them to memory. No timer, no rush.
2. **Get interrupted, on purpose.** Between rounds, you'll solve a couple of quick arithmetic questions. This isn't a math test — it's *interference*, the same kind of mental "noise" that makes real-world memory harder to hold onto. Getting them right or wrong doesn't matter; it's there to occupy your working memory the way daily life does.
3. **Recall.** Find your original glyphs in a shuffled grid of 25.
4. **Level up.** Pass, and the challenge grows — either more interference questions, more glyphs to remember, or both, depending on the mode.

This progressive build-up is the whole idea: short-term memory and working memory get sharper the same way muscles do, by gradually increasing the load rather than jumping straight to the hardest version. A few extra seconds of arithmetic interference, one more glyph to track — small increases, repeated, add up.

## Modes

- **Progressive** — starts small and adds one new glyph every few levels, so the memory load grows steadily as you go.
- **More Progressive** — a bigger jump for people who want a faster ramp and a tougher long-term challenge.

Both are approachable at the start and challenging by the end — good for a quick memory check, for regular practice over time, or as a low-stress cognitive exercise.

---

## Who it's for

- Anyone curious how sharp their short-term memory is right now.
- Anyone who wants a repeatable brain training exercise to come back to.
- Anyone looking for a gentle, self-paced cognitive exercise — no timers, no penalties for wrong math answers, plain language throughout.

Glyph Recall is built for general wellness and entertainment. It's not a medical device and doesn't provide medical advice, diagnosis, or treatment.
